# fullPage.jsを使ったWebページ

**fullPage.jsを使って４つのページがあるWebページを作りました。** <br>
**フォームを入力し、簡単なインタラクションが楽しめます。よろしくお願いします。** 
<br>

**３ページ（third）**
<img width="1433" alt="스크린샷 2024-08-01 오후 2 51 17" src="https://github.com/user-attachments/assets/853cfd02-e513-486a-ba74-9f8331039440">
第３ページから、簡単なインタラクションを楽しめます。<br>
１つ目のコンセプトは『clickでオレンジ色の四角形を生成する』です。javascriptのcanvas機能を使って、マウスでclickしたらキャンバスに四角形を作ることができます。
<img width="1436" alt="스크린샷 2024-08-01 오후 2 51 28" src="https://github.com/user-attachments/assets/178b34a6-2a15-4c57-85a6-4f146dda8e7e">
2つ目のコンセプトは『hoverで四角形の色を変える』です。１つ目とつながる感じで、マウスで四角形にhoverしたらオレンジ色の四角形が黄色の四角形になります。
